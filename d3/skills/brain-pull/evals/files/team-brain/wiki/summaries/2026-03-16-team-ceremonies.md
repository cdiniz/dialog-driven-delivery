# Pageturner Team Ceremonies

## TL;DR
- Agreed two-week sprints with Mon plan / Wed refinements / Fri retro+review cadence.
- QA involved from refinement, not the end. Three amigos required for any story > ½ day.
- Definition of Done locked: merged + CI + staging + tests + sign-offs.
- Standup at **9:40** (moved from 9:30 for [Priya](../people/priya.md)'s school drop-off).

## Key points
- Two refinement sessions per sprint while team is new; drop to one once settled.
- Tech-only session ~30 min once per sprint for tech debt / architecture (no product needed).
- Code review: 1 dev approval; no self-merge; reviewer must actually read.
- Standup format: round-robin yesterday/today/blocked; deep blockers parked for after.
- Retro rotates ([Ben](../people/ben.md) takes the first one).
- Staging is for merge-candidates only. Separate dev environment for experiments.
- UX must stay one sprint ahead of dev — no designs, not refinement-ready.
