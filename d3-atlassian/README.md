# D3 Atlassian Provider Plugin

Atlassian provider for D3 (Dialog Driven Delivery). Enables D3 to work with Confluence for specifications and Jira for user stories.

For full documentation, configuration, and troubleshooting, see the [main README](../README.md).

## Provider Skills

- **atlassian-spec-provider** — Confluence specifications
- **atlassian-story-provider** — Jira user stories
- **atlassian-transcript-provider** — Confluence meeting transcripts

Each skill's `SKILL.md` contains the canonical operation reference and configuration format.
