# D3 Markdown Provider Plugin

Local-first provider for D3 (Dialog Driven Delivery). Store specifications and user stories as markdown files in your git repository.

For full documentation, configuration, and troubleshooting, see the [main README](../README.md).

## Provider Skills

- **markdown-spec-provider** — Local markdown specifications
- **markdown-story-provider** — Local markdown user stories
- **markdown-transcript-provider** — Local markdown meeting transcripts

Each skill's `SKILL.md` contains the canonical operation reference and configuration format.
